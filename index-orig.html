日期组件

  • 基本介绍：

    本系列组件包含时、日、周、月、季、年及时间段等独立选择组件，具有各自的时间选择功能

    注：单击图片跳转到对应demo页面

    依赖文件引入： CSS <link href="../bootstrap-3.3.7-dist/css/bootstrap.css" rel="stylesheet"> <link rel="stylesheet" type="text/css" href="date-all-min.css"> JS <script src="../bootstrap-3.3.7-dist/jquery-1.12.4.min.js"></script> <script src="../js/moment.min.js"></script> <script src="date-all.js"></script>

  • 时间选择：

  • 使用示例

    自定义标签： <time data-id="time" data-auto-fill='true' data-fill='true' data-today-btn='true' data-change-date='changed'></time>

    • 属性
    • 含义
    • 默认值

    • data-id

      指定input的id

      -

    • data-format

      显示格式

      yyyy-mm-dd hh:ii

    • data-minute-step

      "分"界面每个选项间的时间间隔

      5

    • data-fill

      是否启用自动填充

      false

    • data-auto-fill

      是否启用输入时自动填充连接符

      false

    • data-readonly

      是否让输入框只读

      false

    • data-start-date

      开始时间

      2000-01-01

    • data-today-btn

      是否显示底部按钮

      false

    • data-change-date

      改变时间回调

      -

    js调用：

    $('.box').pickDate({
      type: 'time',    //控件类型：时
      id: 'ctime',   //指定input的id
      minuteStep: 1,   //分选择界面每个选项间的时间间隔，单位为 “分”，默认“5”
      format:  ,   //显示格式
      fill: true,    //是否启用自动填充
      readonly: true,    //是否让输入框只读
      startDate: new Date(),   //开始时间
      todayBtn: true,    //是否显示底部按钮
      changeDate: function(){    //改变时间回调
        $showbox.html('已选时间：'+$('#ctime').val())
      }
    })

  • 月范围选择：

  • 使用示例

    自定义标签： <monthrange data-id="monthrange" data-auto-fill='true' data-fill='true' data-readonly='true' data-begin-id='startM' data-end-id='endM' data-today-btn='true' data-change-begin='changed()' data-change-end='changed'></monthrange>

    • 属性
    • 含义
    • 默认值

    • data-begin-id

      指定开始月份input的id

      -

    • data-end-id

      指定结束月份input的id

      -

    • data-begin-class

      指定开始月份div的class

      -

    • data-end-class

      指定结束月份div的class

      -

    • data-fill

      是否启用自动填充

      false

    • data-auto-fill

      是否启用输入时自动填充连接符

      false

    • data-format

      显示格式

      yyyy-mm

    • data-today-btn

      是否显示底部按钮

      false

    • data-readonly

      是否让输入框只读

      false

    • data-change-begin

      改变开始月份回调

      -

    • data-change-end

      改变结束月份回调

      -

    js调用：

    $('.box').pickDate({
      type: 'monthRange',    //控件类型：月范围
      beginClass: 'startime',    //指定开始时间div的class
      beginId: 'startMonth',   //指定开始时间input的id
      endClass: 'endtime',   //指定结束时间div的class
      endId: 'endMonth',   //指定结束时间input的id
      fill: true,    //是否启用自动填充
      changeBegin: function(){   //改变开始时间回调
        $showbox.html('已选时间：'+$('#startMonth').val())
      },
      changeEnd: function(){   //改变结束时间回调
        $showbox.html('已选时间：'+$('#endMonth').val())
      }
    })
